**Nota para chatbot IA (importante)**

- Esta página se usa **solo** cuando el usuario indica que **un color concreto no tiene stock** (o que necesita alternativa inmediata).
- Objetivo: proponer **alternativas de color** dentro de colecciones de **calidad/uso similares**, priorizando las que sí tengan stock.
- Si preguntan por **colecciones aptas o recomendadas** para un requisito (p. ej., **suelo radiante** o **mascotas**), buscar primero en la página de **colecciones recomendadas**, donde está el desglose completo.
